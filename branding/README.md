# Branding Direction

Nassim Systems is positioned as a premium AI consulting studio with a calm, high-trust visual identity.

## Core Palette
- Navy: #07111F
- Deep Blue: #0F274A
- Cyan Accent: #00D4FF
- Slate: #1C2A3A

## Brand Concept
This visual system is designed to feel:
- authoritative
- modern
- intelligent
- premium
- technically elevated
